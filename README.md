# ChroniSync

## Project Brief

ChroniSync is an AI-assisted, guideline-grounded chronic disease monitoring platform connecting patients, caregivers, and physicians across four conditions — type 2 diabetes, hypertension, chronic kidney disease, and COPD. Patients log vitals, medications, symptoms, and daily check-ins; the system contextualizes that data against disease-specific clinical guidelines and generates AI summaries, trend explanations, and rule-based alerts for both patient and doctor. Every AI output is grounded in a retrieved, disease-specific knowledge pack rather than the model's unconstrained knowledge, keeping the platform a transparent, clinician-reviewable coordination layer — not a diagnosis engine.

## Google Technology Used

- **Gemini API** — AI-assisted document extraction and visit/trend summarization
- **Firebase Authentication** — sign-in, registration, password reset, and role-based session routing
- **Cloud Firestore** — structured storage for clinical and portal data
- **Firebase Storage** — uploaded patient documents and files
- **Firebase Admin SDK** — secure server-side access to Firebase services
- **Google Cloud infrastructure** — underlying infrastructure via Firebase-backed services
